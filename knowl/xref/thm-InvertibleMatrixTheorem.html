Theorem 2.4.15. Invertible Matrix Theorem.

The following conditions are equivalent for an \(n \times n\) matrix:
  1. \(A\) is invertible.
    🔗
    🔗
  2. The homogeneous system \(A\v{x} = \v{0}\) has only the trivial solution \(\v{x} = \v{0}\text{.}\)
    🔗
    🔗
  3. \(A\) can be carried to the identity matrix \(I\) by elementary row operations.
    🔗
    🔗
  4. The system \(A\v{x} = \v{b}\) has at least one solution \(\v{x}\) for every choice of column \(\v{b}\text{.}\)
    🔗
    🔗
  5. There exists an \(n \times n\) matrix \(C\) such that \(AC = I\text{.}\)
    🔗
    🔗
  6. \(\rank A = n\text{.}\)
    🔗
