    Worksheet Systems of Linear Equations
    

    1.

    Solve the following system of equations:
    \begin{equation*} \begin{cases} x_1 + 3x_2 - 5x_3 + x_4 = 4 \\ 2x_1 + 5x_2 - 2x_3 + 4x_4 = 6 \\ \end{cases} \end{equation*}
    If the system is consistent, write the solution set as a column vector If the system is inconsistent, explain how you know it.
    🔗
    Answer.
    The solutions are of the form \(\begin{bmatrix} -2 - 19s - 7t \\ 2 + 8s + 2t \\ s \\ t \end{bmatrix}\) where \(s\) and \(t\) are arbitrary constants.

    If the system is consistent, write the solution set as a column vector If the system is inconsistent, explain how you know it.

    3.

    If the system is consistent, write the solution set as a column vector If the system is inconsistent, explain how you know it.
    🔗
    🔗

    4.

    Solve the following system of equations:
    \begin{equation*} \begin{cases} 2x_1 + 4x_2 + 6x_3 = 0 \\ 4x_1 + 5x_2 + 6x_3 = 0 \\ 3x_1 + x_2 - 2x_3 = 0 \end{cases} \end{equation*}
    If the system is consistent, write the solution set as a linear combination of basic vectors. If the system is inconsistent, explain how you know it.
    🔗
    Answer.
    This one has only the trivial solution \(\begin{bmatrix} 0 \\ 0 \\ 0\end{bmatrix}\)
    🔗
    🔗
    🔗

    5.

    Solve the following system of equations:
    \begin{equation*} \begin{cases} x_1 + 2x_2 - x_3 = 0 \\ 3x_1 - 3x_2 + 2x_3 = 0 \\ -x_1 - 11x_2 + 6x_3 = 0 \end{cases} \end{equation*}
    If the system is consistent, write the solution set as a linear combination of basic vectors. If the system is inconsistent, explain how you know it.
    🔗
    Answer.
    The solutions are of the form \(t\begin{bmatrix} -\frac{1}{9} \\ \frac{5}{9} \\ 1 \end{bmatrix}\)
    🔗
    🔗
    🔗

    6.

    Solve the following system of equations:
    \begin{equation*} \begin{cases} 3x + 3y + 2w = 0 \\ y - z = 0 \\ 3x + 5y - 2z + 2w = 0 \end{cases} \end{equation*}
    If the system is consistent, write the solution set as a linear combination of basic vectors. If the system is inconsistent, explain how you know it.
    🔗
    Answer.
    The solutions are of the form \(z\begin{bmatrix} -1 \\ 1 \\ 1 \\ 0\end{bmatrix} + w\begin{bmatrix}-\frac{2}{3} \\ 0 \\ 0 \\ 1 \end{bmatrix}\)
