    Section 1.3 Homogeneous Equations

    In this section, we explore a special type of systems of linear equations, called the homogeneous systems of linear equations.
    🔗

    Objectives

    After this section, students will be able to:
    🔗
    • solve homogeneous systems of linear equations using Gaussian elimination algorithm.
      🔗
      🔗
    • write the general solution of a homogeneous system as a linear combination of basic vectors (written as column vectors).
      🔗
      🔗
    🔗

    Investigation 1.3.1.

    Richard claims that the following system of linear equations has at least one solution (hence a consistent system) at the first glance.
    \begin{equation*} \begin{cases} 4x - 12y + 2z = 0 \\ 2x - 6y = 0 \\ 3x - 9y + z = 0 \\ -2x + 6y - z = 0 \end{cases} \end{equation*}
    Why does Richard know about it quickly?
    🔗
    Hint.
    Recall something is called a solution if it satisfies all the equations in the system. Can you guess-and-check a solution to this system quickly?
    🔗
    🔗
    Solution.
    If you plug in zero for every variable, you will get zero on the left-hand side of every equation and solve all of them! So \((0,0,0)\) is a solution to this system of linear equations.
    🔗
    When you have all zeros for the constant terms, you always have this solution (all variables equal to zero), which is called the "trivial solution".
    🔗
    🔗
    🔗
    In this section, we will learn about a super important special kind of system of equations called homogeneous systems.
    🔗
    A homogeneous system of equations (or a system of homogeneous equations) is just a special case where all of the constants to the right of the equal sign are zero.
    🔗
    For example, we worked with the following system of linear equations in the last section:
    \begin{equation*} \begin{cases} 4x - 12y + 2z = -12 \\ 2x - 6y = -10 \\ 3x - 9y + z = -11 \\ -2x + 6y - z = 6 \end{cases} \end{equation*}
    This is NOT a homogeneous system since the constants on the right-hand side are not all zero. We can make it homogeneous by changing the number on the right-hand side of every equation to a zero as follows:
    \begin{equation*} \begin{cases} 4x - 12y + 2z = 0 \\ 2x - 6y = 0 \\ 3x - 9y + z = 0 \\ -2x + 6y - z = 0 \end{cases} \end{equation*}
    Then we have a homogeneous system!
    🔗
    Note: Just to be clear, this is a different system of linear equations. However, we will see later that it is actually related to the original one in an important way!
    🔗
    Usually, a system of linear equations can have either (1) zero, (2) one, or (3) infinitely many solutions. But it is impossible for a homogeneous system to have zero solutions! As we figured out the investigation question, every homogeneous system has at least the trivial solution (where all the variables are zero). So a homogeneous system either has one solution (trivial) OR it has infinitely many solutions.
    🔗
    Let’s solve our homogeneous system in the investigation! First, we will write the augmented matrix for the system:
    \begin{equation*} \begin{bmatrix} 4 \amp -12 \amp 2 \amp \mid \amp 0 \\ 2 \amp -6 \amp 0 \amp \mid \amp 0 \\ 3 \amp -9 \amp 1 \amp \mid \amp 0 \\ -2 \amp 6 \amp -1 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    We can actually do the exact same steps that we did with the non-homogeneous version. The only difference is that the constant column doesn’t require any work since all the entries hold steady at zero.
    🔗
    So if we put it into row echelon form, we get:
    \begin{equation*} \begin{bmatrix} 1 \amp -3 \amp \frac{1}{2} \amp \mid \amp 0 \\ 0 \amp 0 \amp 1 \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    Now we can convert back to a system of linear equations and we get:
    \begin{equation*} \begin{cases} x - 3y + \frac{1}{2}z = 0 \\ z = 0 \\ 0 = 0 \\ 0 = 0 \end{cases} \end{equation*}
    Just like with the regular version, \(x\) and \(z\) are the leading variables and \(y\) is a free variable. We can back substitute \(z = 0\) into the first equation to get \(x = 3y\) and write the solution in parametric form:
    \begin{equation*} \begin{cases} x = 3y \\ y = y \\ z = 0 \end{cases} \end{equation*}
    or by writing what an arbitrary point in the solution set looks like: \((3y, y, 0)\)
    🔗

    Subsection Basis Vectors and Linear Combinations

    Now, notice that we can factor out a \(y\) in the solution and see that the set of solutions is of the form
    \begin{equation*} y(3,1,0) \end{equation*}
    where \(y\) can be any real number. That means every solution of the system is a multiple of \((3,1,0)\text{.}\)
    🔗
    We call this \((3,1,0)\) a "basis" or "basic vector" of the solution set (Actually "basis" is much more common, but the author uses "basic" in the textbook...).
    🔗

    Example 1.3.1.

    Pretend that we had a (homogeneous) system of 4 linear equations in 5 variables \((x,y,z,u,z)\) and we put the augmented matrix into row echelon form and got:
    \begin{equation*} \begin{bmatrix} 1 \amp -3 \amp 5 \amp 2 \amp 7 \amp \mid \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \amp 1 \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    The leading variables are \(x\text{,}\) \(z\text{,}\) and \(v\text{,}\) so our free variables are \(y\) and \(u\text{.}\) That is, we should be able to solve for the other three variables in terms of \(y\) and \(u\text{.}\)
    🔗
    Let’s convert it to a system of linear equations:
    \begin{equation*} \begin{cases} x - 3y + 5z + 2u + 7v = 0 \\ z + v = 0 \\ v = 0 \\ 0 = 0 \end{cases} \end{equation*}
    We know that \(v = 0\) and we can back substitute into the second equation to get \(z = 0\) as well.
    🔗
    Back substituting into the first equation, we get
    \begin{equation*} x - 3y + 2u = 0 \qquad \implies \qquad x = 3y - 2u \end{equation*}
    Now we can write an arbitrary point in the solution set as
    \begin{equation*} (3y - 2u, y, 0, u, 0) \end{equation*}
    Now we can split this up into two vectors, one with only \(y\)’s and one with only \(u\)’s:
    \begin{align*} (3y - 2u, y, 0, u, 0) \amp= (3y,y,0,0,0) + (-2u,0,0,u,0) \\ \amp= y(3,1,0,0,0) + u(-2,0,0,1,0) \amp\amp \text{factor out the $y$ and $u$} \end{align*}
    This means that every solution is a linear combination of the "basic" solutions \((3,1,0,0,0)\) and \((-2,0,0,1,0)\text{.}\)
    🔗
    🔗
    Points in \(n\)-dimensional space are represented as ordered \(n\)-tuples. For example, the point \((2,4,5,1)\) is a point in \(4\)-space (we represent \(4\)-space by \(\R^4\)). In linear algebra, such points can also be represented as \(1 \times n\) matrices (row vectors) or \(n \times 1\) matrices (column vectors). So \((2,4,5,1)\) can be represented as the row vector
    \begin{equation*} \begin{bmatrix} 2 \amp 4 \amp 5 \amp 1 \end{bmatrix} \end{equation*}
    or the column vector
    \begin{equation*} \begin{bmatrix} 2 \\ 4 \\ 5 \\ 1 \end{bmatrix} \end{equation*}
    In this class, we will mostly use column vectors for reasons that will become clear in Section 2.2.
    🔗

    Example 1.3.2.

    Recall that we did our earlier example we got the solution
    \begin{equation*} (3y - 2u, y, 0, u, 0) \end{equation*}
    which we broke up into the sum of two points like this:
    \begin{equation*} (3y,y,0,0,0) + (-2u,0,0,u,0) \end{equation*}
    Then we factor out the \(y\) and \(u\) to get
    \begin{equation*} y(3,1,0,0,0) + u(-2,0,0,1,0) \end{equation*}
    🔗
    Now, we can rewrite all of this using column vectors! An arbitrary solution (in parametric form) would look like
    \begin{align*} \begin{bmatrix} 3y - 2u \\ y \\ 0 \\ u \\ 0 \end{bmatrix} \amp= \begin{bmatrix} 3y \\ y \\ 0 \\ 0 \\ 0 \end{bmatrix} + \begin{bmatrix} -2u \\ 0 \\ 0 \\ u \\ 0 \end{bmatrix} \\ \amp= y \begin{bmatrix} 3 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix} + u \begin{bmatrix} -2 \\ 0 \\ 0 \\ 1 \\ 0 \end{bmatrix} \end{align*}
    Notice that any vector of this form is a solution to the homogeneous system of linear equations. The \(y\) and \(u\) can be any number!
    🔗
    In particular, this means that both \(\begin{bmatrix}3 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix}\) and \(\begin{bmatrix}-2 \\ 0 \\ 0 \\ 1 \\ 0 \end{bmatrix}\) are solutions! That is one reason why they are called the basic solutions (it explains the "solution" part). We will learn more about wht they are called "basic" in the future!
    🔗
    🔗
    To make sure we are all on the same page, recall we define two operations on vectors: vector addition and scalar multiplication. They are both defined component-wise. That is,
    • to add two vectors, we just add together the corresponding entries, and
      🔗
      🔗
    • to multiply a vector by a scalar, we just multiply every entry by that scalar.
      🔗
      🔗
    🔗
    Now let’s talk about linear combinations! A linear combination of a set of vectors is a sum of scalar multiples of those vectors.
    🔗

    Example 1.3.3.

    1. \((2,3)\) is a linear combination of \((1,0)\) and \((0,1)\) since
      \begin{equation*} (2,3) = 2(1,0) + 3(0,1) \end{equation*}
      🔗
      🔗
    2. \((2,3)\) is also a linear combination of \((1,0)\) and \((1,1)\) since
      \begin{equation*} (2,3) = -1(1,0) + 3(1,1) \end{equation*}
      🔗
      🔗
    🔗
    The expression
    \begin{equation*} y \begin{bmatrix} 3 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix} + u \begin{bmatrix} -2 \\ 0 \\ 0 \\ 1 \\ 0 \end{bmatrix} \end{equation*}
    represents the set of all possible linear combinations of \(\begin{bmatrix} 3 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix}\) and \(\begin{bmatrix} -2 \\ 0 \\ 0 \\ 1 \\ 0 \end{bmatrix}\text{.}\)
    🔗

    Checkpoint 1.3.4.

    Solve the following homogeneous system of linear equations and write an arbitrary solution as a linear combination of basic solutions (written as column vectors):
    \begin{equation*} \begin{cases} 3x - 2y + z + 5u = 0 \\ x - 6y + 2z + u = 0 \\ x + 10y - 3z + 3u = 0 \\ 4x - 8y + 3z + 6u = 0 \end{cases} \end{equation*}
    🔗
    Solution.
    Let’s start by writing the augmented matrix for the system:
    \begin{equation*} \begin{bmatrix} 3 \amp -2 \amp 1 \amp 5 \amp \mid \amp 0 \\ 1 \amp -6 \amp 2 \amp 1 \amp \mid \amp 0 \\ 1 \amp 10 \amp -3 \amp 3 \amp \mid \amp 0 \\ 4 \amp -8 \amp 3 \amp 6 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    We will solve the system using the Gaussian elimination algorithm.
    🔗
    Step 1: Swap Row 1 and Row 2 so that we get a leading \(1\) without using any fractions:
    \begin{equation*} \begin{bmatrix} 1 \amp -6 \amp 2 \amp 1 \amp \mid \amp 0 \\ 3 \amp -2 \amp 1 \amp 5 \amp \mid \amp 0 \\ 1 \amp 10 \amp -3 \amp 3 \amp \mid \amp 0 \\ 4 \amp -8 \amp 3 \amp 6 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    🔗
    Step 2: Add \(-3\) times Row 1 to Row 2 (and replace Row 2), add \(-1\) times Row 1 to Row 3 (and replace Row 3), and add \(-4\) times Row 1 to Row 4 (and replace Row 4) to get:
    \begin{equation*} \begin{bmatrix} 1 \amp -6 \amp 2 \amp 1 \amp \mid \amp 0 \\ 0 \amp 16 \amp -5 \amp 2 \amp \mid \amp 0 \\ 0 \amp 16 \amp -5 \amp 2 \amp \mid \amp 0 \\ 0 \amp 16 \amp -5 \amp 2 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    🔗
    Step 3: Multiply Row 2, Row 3, and Row 4 by \(\frac{1}{16}\) (observe the last three rows are exactly the same!) to get:
    \begin{equation*} \begin{bmatrix} 1 \amp -6 \amp 2 \amp 1 \amp \mid \amp 0 \\ 0 \amp 1 \amp -\frac{5}{16} \amp \frac{1}{8} \amp \mid \amp 0 \\ 0 \amp 1 \amp -\frac{5}{16} \amp \frac{1}{8} \amp \mid \amp 0 \\ 0 \amp 1 \amp -\frac{5}{16} \amp \frac{1}{8} \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    🔗
    Step 4: Add \(-1\) times Row 2 to Row 3 (and replace Row 3) and add \(-1\) times Row 2 to Row 4 (and replace Row 4) to get:
    \begin{equation*} \begin{bmatrix} 1 \amp -6 \amp 2 \amp 1 \amp \mid \amp 0 \\ 0 \amp 1 \amp -\frac{5}{16} \amp \frac{1}{8} \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    🔗
    We can see from the leading \(1\)s that \(x\) and \(y\) are the leading variables and \(z\) and \(u\) are the free variables. We can also see this from converting back to a system of linear equations:
    \begin{equation*} \begin{cases} x - 6y + 2z + u = 0 \\ y - \frac{5}{16}z + \frac{1}{8}u = 0 \\ 0 = 0 \\ 0 = 0 \end{cases} \end{equation*}
    We can solve for \(x\) in terms of \(y\text{,}\) \(z\text{,}\) and \(u\) to get:
    \begin{equation*} x = 6y - 2z - u \end{equation*}
    We can also solve for \(y\) in terms of \(z\) and \(u\) to get:
    \begin{equation*} y = \frac{5}{16}z - \frac{1}{8}u \end{equation*}
    Then we can back substitute this expression for \(y\) into the first equation to get:
    \begin{align*} x \amp= 6 \lp \frac{5}{16}z - \frac{1}{8}u \rp - 2z - u \\ \amp- \frac{15}{8}z - \frac{3}{4}u - 2z - u \\ \amp= -\frac{1}{8}z - \frac{7}{4}u \end{align*}
    Now let’s write our general solution as a column vector! Every solution looks like
    \begin{align*} \begin{bmatrix} -\frac{1}{8}z - \frac{7}{4}u \\ \frac{5}{16}z - \frac{1}{8}u \\ z \\ u \end{bmatrix} \amp= \begin{bmatrix} -\frac{1}{8}z \\ \frac{5}{16}z \\ z \\ 0 \end{bmatrix} + \begin{bmatrix} -\frac{7}{4}u \\ -\frac{1}{8}u \\ 0 \\ u \end{bmatrix} \\ \amp= z \begin{bmatrix} -\frac{1}{8} \\ \frac{5}{16} \\ 1 \\ 0 \end{bmatrix} + u \begin{bmatrix} -\frac{7}{4} \\ -\frac{1}{8} \\ 0 \\ 1 \end{bmatrix} \end{align*}
    🔗
    Note 1.3.5. If you want to turn the augmented matrix into reduced row echelon form....
    To get the augmented matrix into reduced row echelon form, we will just need to do one more step after Step 4.
    🔗
    Step 5: Add \(6\) times Row 2 to Row 1 (and replace Row 1) to get:
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp \frac{1}{8} \amp \frac{7}{4} \amp \mid \amp 0 \\ 0 \amp 1 \amp -\frac{5}{16} \amp \frac{1}{8} \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    🔗
    Converting it back to the system of linear equations, we get:
    \begin{equation*} \begin{cases} x + \frac{1}{8}z + \frac{7}{4}u = 0 \\ y - \frac{5}{16}z + \frac{1}{8}u = 0 \\ 0 = 0 \\ 0 = 0 \end{cases} \end{equation*}
    Similarly, we see that \(z\) and \(u\) are the free variables, and hence we will represent \(x\) and \(y\) in terms of \(z\) and \(u\text{:}\)
    \begin{equation*} \begin{cases} x = -\frac{1}{8}z - \frac{7}{4}u \\ y = \frac{5}{16}z - \frac{1}{8}u \end{cases} \end{equation*}
    🔗
    Then we can write our general solution as a column vector! Every solution looks like
    \begin{equation*} \begin{bmatrix} -\frac{1}{8}z - \frac{7}{4}u \\ \frac{5}{16}z - \frac{1}{8}u \\ z \\ u \end{bmatrix} = z \begin{bmatrix} -\frac{1}{8} \\ \frac{5}{16} \\ 1 \\ 0 \end{bmatrix} + u \begin{bmatrix} -\frac{7}{4} \\ -\frac{1}{8} \\ 0 \\ 1 \end{bmatrix} \end{equation*}
    🔗
    🔗
    🔗
    🔗
    We are actually exposed to some big concepts in linear algebra, like linear combinations and basic vectors. We will loop back to these concepts formally later when we get to the theory of vector spaces. For now, it is totally fine if you have only a vague understanding of these concepts. Remember the overall goal of this section is to learn how to solve homogeneous systems of linear equations!
