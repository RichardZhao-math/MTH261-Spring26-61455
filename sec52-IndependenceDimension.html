    Section 5.2 Independence and Dimension

    In the previous section, we explored how to build a subspace using a spanning set. But this raises an important question for us to investigate: are all the vectors in a spanning set actually necessary? Sometimes, a set contains "redundant" vectors that don’t add any new direction to the space.
    🔗
    In this section, we are going to explore the concepts of linear independence and dimension.
    🔗

    Objectives

    After this section, students will be able to:
    🔗
    • determine if a set of vectors are linearly independent.
      🔗
      🔗
    • construct a basis for a subspace.
      🔗
      🔗
    • determine the dimension of a subspace.
      🔗
      🔗
    🔗

    Subsection Linear Independence

    When building a span, are all the vectors actually necessary? Sometimes a set contains redundant vectors that don’t contribute to the span. Let’s keep this in mind as we investigate the concept of linear independence.
    🔗

    Definition 5.2.1.

    An indexed set of vectors \(\set{\v{v}_1, \v{v}_2, \ldots, \v{v}_p}\text{,}\) \(\v{v}_i\in \R^n\text{,}\) is linearly independent if the vector equation
    \begin{equation*} x_1\v{v}_1 + x_2\v{v}_2 + \cdots + x_p\v{v}_p = \v{0} \end{equation*}
    has only the trivial solution.
    🔗
    If the trivial solution is not unique, we call the set linear dependent.
    🔗
    That is, if a set \(\set{\v{v}_1, \v{v}_2, \ldots, \v{v}_p}\text{,}\) \(\v{v}_i\in \R^n\text{,}\) is linearly dependent, then there exists some scalars \(c_1, c_2, \ldots, c_p\in \R\text{,}\) not all zero, such that
    \begin{equation*} x_1\v{v}_1 + x_2\v{v}_2 + \cdots + x_p\v{v}_p = \v{0} \end{equation*}
    🔗
    🔗
    But how does the vector equation even related to the idea of independence? We will demonstrate the relation in the following example.
    🔗

    Example 5.2.2.

    Suppose we have a set of three vectors \(\set{\v{v}_1, \v{v}_2, \v{v}_3}\text{,}\) where
    \begin{equation*} \v{v}_1 = \begin{bmatrix} 1 \\ 2 \\ 1\end{bmatrix}\, , \qquad \v{v}_2 = \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix}\, , \qquad \text{ and } \, \v{v}_3 = \begin{bmatrix} 3 \\ 5 \\ 3\end{bmatrix} \end{equation*}
    Is this set of vectors linearly independent?
    🔗
    Let’s follow the definition! That is, we want to determine if the solution to the following vector equation
    \begin{equation*} x_1 \begin{bmatrix} 1 \\ 2 \\ 1\end{bmatrix} + x_2 \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix} + x_3 \begin{bmatrix} 3 \\ 5 \\ 3\end{bmatrix} = \begin{bmatrix} 0 \\ 0 \\ 0 \end{bmatrix} \end{equation*}
    is unique.
    🔗
    We know how to solve this equation! We will first convert it to an augmented matrix:
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 3 \amp \mid \amp 0 \\ 2 \amp 1 \amp 5 \amp \mid \amp 0 \\ 1 \amp 0 \amp 3 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    Now let’s row reduce this augmented matrix!
    • Step 1: Adding \(-2\) times Row 1 to Row 2 (and replacing Row 2), we obtain
      \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 3 \amp \mid \amp 0 \\ 0 \amp 1 \amp -1 \amp \mid \amp 0 \\ 1 \amp 0 \amp 3 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
      🔗
      🔗
    • Step 2: Adding \(-1\) times Row 1 to Row 3 (and replacing Row 3), we obtain
      \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 3 \amp \mid \amp 0 \\ 0 \amp 1 \amp -1 \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
      🔗
      🔗
    We immediately know that \(x_3\) is a free variable since there is no pivot position in the third column. Hence, this vector equation has infinitely many solutions.
    🔗
    By the definition, we know that \(\set{\v{v}_1, \v{v}_2, \v{v}_3}\) is linearly dependent.
    🔗
    But what does it mean? Why do we call these three vectors linearly dependent? Well let’s finish solving this vector equation!
    🔗
    Since \(x_3\) is a free variable, then we let \(x_3 = t\text{,}\) where \(t\in \R\text{.}\) Then we obtain
    \begin{equation*} \begin{cases} x_1 = -3x_3 = -3t \\ x_2 = x_3 = t \\ x_3 = t \end{cases} \end{equation*}
    Okay. This is a general solution to the vector equation. Recall we can obtain a particular solution by assigning a value to our parameter, \(t\text{.}\)
    🔗
    Let’s make \(t = 1\text{.}\) Then our particular solution is
    \begin{equation*} \begin{cases} x_1 = -3(1) = -3 \\ x_2 = 1 \\ x_3 = 1 \end{cases} \end{equation*}
    Then plugging in the solution to the vector equation will make the equation true. That is, we have the following equation:
    \begin{equation*} -3 \begin{bmatrix} 1 \\ 2 \\ 1\end{bmatrix} + 1 \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix} + 1 \begin{bmatrix} 3 \\ 5 \\ 3\end{bmatrix} = \begin{bmatrix} 0 \\ 0 \\ 0 \end{bmatrix} \end{equation*}
    We can isolate one of the three vectors! Just for demonstration purposes, let’s isolate \(\v{v}_2\text{!}\) Then we have
    \begin{equation*} \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix} = 3 \begin{bmatrix} 1 \\ 2 \\ 1\end{bmatrix} - \begin{bmatrix} 3 \\ 5 \\ 3\end{bmatrix} \end{equation*}
    Observe that \(\v{v}_2\) is a linear combination of \(\v{v}_1\) and \(\v{v}_3\text{!}\) That is, \(\v{v}_2\) is dependent on \(\v{v}_1\) and \(\v{v}_3\text{.}\)
    🔗
    🔗
    The practical meaning of a linearly dependent set of vectors is that a vector can be expressed as a linear combination of the other vectors in the set. That is, this vector is dependent of other vectors. This specific relation is called the linear dependence relation.
    🔗
    In the above example, a linear dependence relation is
    \begin{equation*} -3 \begin{bmatrix} 1 \\ 2 \\ 1\end{bmatrix} + 1 \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix} + 1 \begin{bmatrix} 3 \\ 5 \\ 3\end{bmatrix} = \begin{bmatrix} 0 \\ 0 \\ 0 \end{bmatrix} \end{equation*}
    since this relation allows us to express a vector, \(\v{v}_2\text{,}\) as a linear combination of the other vectors, \(\v{v}_1\) and \(\v{v}_3\text{.}\)
    🔗

    Checkpoint 5.2.3.

    Is the set \(\left\{\begin{bmatrix} 1 \\ 1 \\ 1\end{bmatrix}\, , \, \begin{bmatrix} 0 \\ 1 \\ 2 \end{bmatrix}\, , \, \begin{bmatrix} 1 \\ 2 \\ 0\end{bmatrix}\right\}\) linearly independent?
    🔗
    Solution.
    Yes, this set of vectors is linearly independent.
    🔗
    To justify this, we set up the homogeneous vector equation:
    \begin{equation*} x_1 \begin{bmatrix} 1 \\ 1 \\ 1\end{bmatrix} + x_2 \begin{bmatrix} 0 \\ 1 \\ 2 \end{bmatrix} + x_3 \begin{bmatrix} 1 \\ 2 \\ 0\end{bmatrix} = \begin{bmatrix} 0 \\ 0 \\ 0 \end{bmatrix} \end{equation*}
    🔗
    We can solve this by row reducing the corresponding augmented matrix:
    \begin{align*} \begin{bmatrix} 1 \amp 0 \amp 1 \amp \mid \amp 0 \\ 1 \amp 1 \amp 2 \amp \mid \amp 0 \\ 1 \amp 2 \amp 0 \amp \mid \amp 0 \end{bmatrix} \amp\sim \begin{bmatrix} 1 \amp 0 \amp 1 \amp \mid \amp 0 \\ 0 \amp 1 \amp 1 \amp \mid \amp 0 \\ 0 \amp 2 \amp -1 \amp \mid \amp 0 \end{bmatrix} \amp\amp \begin{matrix}-\text{R1} + \text{R2} \\ -\text{R1} + \text{R3} \end{matrix}\\ \amp\sim \begin{bmatrix} 1 \amp 0 \amp 1 \amp \mid \amp 0 \\ 0 \amp 1 \amp 1 \amp \mid \amp 0 \\ 0 \amp 0 \amp -3 \amp \mid \amp 0 \end{bmatrix} \amp\amp -2\text{R2} + \text{R3} \\ \amp\sim \begin{bmatrix} 1 \amp 0 \amp 1 \amp \mid \amp 0 \\ 0 \amp 1 \amp 1 \amp \mid \amp 0 \\ 0 \amp 0 \amp 1 \amp \mid \amp 0 \end{bmatrix} \amp\amp \frac{1}{-3}\text{R3} \end{align*}
    🔗
    Notice that every column in the coefficient matrix contains a pivot position, which means there are no free variables. Therefore, the vector equation has only the trivial solution (\(x_1=0, x_2=0, x_3=0\)). Because no vector can be built from the others, the set is linearly independent.
    🔗
    🔗
    🔗
    You may notice that the procedure to determine whether a set of column vectors are linearly independent is essentially the same as row reducing an augmented matrix representing a homogeneous system of linear equations. That is, we are essentially solving the homogeneous matrix equation \(A\v{x} = \v{0}\text{,}\) where \(A\) consists of the vectors of interest.
    • If \(A\v{x} = \v{0}\) has only the trivial solution, then the columns of \(A\) are linearly independent;
      🔗
      🔗
    • If \(A\v{x} = \v{0}\) has infinitely many solutions, then the columns of \(A\) are linearly dependent.
      🔗
      🔗
    🔗

    Example 5.2.4.

    Suppose we are given the coefficient matrix \(A = \begin{bmatrix} 1 \amp 1 \amp 2 \\ 2 \amp 3 \amp 5 \\ 1 \amp 0 \amp 3 \end{bmatrix}\text{.}\) Observe that \(A\) consists of three column vectors:
    \begin{equation*} \v{v}_1 = \begin{bmatrix} 1 \\ 2 \\ 1 \end{bmatrix}\, , \qquad \v{v}_2 = \begin{bmatrix} 1 \\ 3 \\ 0 \end{bmatrix}\, , \qquad \text{ and } \, \v{v}_3 = \begin{bmatrix} 2 \\ 5 \\ 3 \end{bmatrix} \end{equation*}
    Are they linearly independent?
    🔗
    We can answer this question by solving the homogeneous matrix equation \(A\v{x} = \v{0}\text{.}\) The augmented matrix is
    \begin{equation*} \begin{bmatrix} 1 \amp 1 \amp 2 \amp \mid \amp 0 \\ 2 \amp 3 \amp 5 \amp \mid \amp 0 \\ 1 \amp 0 \amp 3 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    Of course we know how to row reduce it! Notice that the constant column of zeros will stay unchanged after all the elementary row operations. So we can only row reduce the coefficient matrix part!
    \begin{align*} \begin{bmatrix} 1 \amp 1 \amp 2 \\ 2 \amp 3 \amp 5 \\ 1 \amp 0 \amp 3 \end{bmatrix} \amp\sim \begin{bmatrix} 1 \amp 1 \amp 2 \\ 0 \amp 1 \amp 1 \\ 1 \amp 0 \amp 3 \end{bmatrix} \amp\amp -2\text{R1} + \text{R2} \\ \amp\sim \begin{bmatrix} 1 \amp 1 \amp 2 \\ 0 \amp 1 \amp 1 \\ 0 \amp -1 \amp 1 \end{bmatrix} \amp\amp -\text{R1} + \text{R3} \\ \amp\sim \begin{bmatrix} 1 \amp 0 \amp 1 \\ 0 \amp 1 \amp 1 \\ 0 \amp -1 \amp 1 \end{bmatrix} \amp\amp -\text{R2} + \text{R1} \\ \amp\sim \begin{bmatrix} 1 \amp 0 \amp 1 \\ 0 \amp 1 \amp 1 \\ 0 \amp 0 \amp 2 \end{bmatrix} \amp\amp \text{R2} + \text{R3} \\ \amp\sim \begin{bmatrix} 1 \amp 0 \amp 1 \\ 0 \amp 1 \amp 1 \\ 0 \amp 0 \amp 1 \end{bmatrix} \amp\amp \frac{1}{2}\text{R3} \\ \amp\sim \begin{bmatrix} 1 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} \amp\amp -\text{R3} + \text{R2} \\ \amp\sim \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} \amp\amp -\text{R3} + \text{R1} \end{align*}
    Recall the constant column will consist of all zeros.
    🔗
    Observe that every column of the coefficient matrix has a pivot position, which implies that there is no free variable. Then this homogeneous matrix equation has only the trivial solution. That is, the only "linearly dependent relation" is
    \begin{equation*} 0 \begin{bmatrix} 1 \\ 2 \\ 1 \end{bmatrix} + 0 \begin{bmatrix} 1 \\ 3 \\ 0 \end{bmatrix} + 0 \begin{bmatrix} 2 \\ 5 \\ 3 \end{bmatrix} = \begin{bmatrix} 0 \\ 0 \\ 0 \end{bmatrix} \end{equation*}
    Well we can’t isolate any of the three vectors since all the above relation tells us is that \(\v{0} = \v{0}\text{.}\) Hence, it makes sense to call the vectors making up the columns of \(A\) linearly independent, in the sense that none of the vector is dependent of other vectors.
    🔗
    🔗
    It turns out that there are some special theorems to determine whether a set of vectors are linearly independent. Let’s look at the following example and see if we can generalize the theorems!
    🔗

    Example 5.2.5.

    Is \(\left\{\begin{bmatrix} 1 \\ 2 \\ 3 \end{bmatrix}, \begin{bmatrix} 2 \\ 4 \\ 6\end{bmatrix}\right\}\) linearly independent?
    🔗
    Think about it first and see if you can figure it out. Then expand the following solution to check your answer!
    🔗
    Solution.
    These two vectors are linearly dependent since there is a way to write a vector as a linear combination of the other vector.
    🔗
    Clearly, \(\begin{bmatrix} 2 \\ 4 \\ 6\end{bmatrix} = 2 \begin{bmatrix} 1 \\ 2 \\ 3\end{bmatrix}\text{.}\) That is, the vector \(\begin{bmatrix} 2 \\ 4 \\ 6\end{bmatrix} \) is a linear combination of the vector \(\begin{bmatrix} 1 \\ 2 \\ 3\end{bmatrix} \text{.}\) So \(\begin{bmatrix} 2 \\ 4 \\ 6\end{bmatrix} \) is dependent of \(\begin{bmatrix} 1 \\ 2 \\ 3\end{bmatrix} \text{.}\)
    🔗
    🔗
    🔗
    Using the similar argument, we can make sense of the following theorem:
    🔗

    Theorem 5.2.6.

    Consider \(\set{\v{v}_1, \v{v}_2}\text{.}\) If \(\v{v}_2 = c\v{v}_1\) for some nonzero constant \(c\text{,}\) then \(\v{v}_1\) and \(\v{v}_2\) are linearly dependent.
    🔗
    🔗
    Now let’s look at the next example! Remember the goal is to generalize it!
    🔗

    Example 5.2.7.

    Is \(\left\{\begin{bmatrix} 1 \\ 2 \\ 3 \end{bmatrix}, \begin{bmatrix} 1 \\ 5 \\ 6\end{bmatrix}, \begin{bmatrix} 0 \\ 0 \\ 0 \end{bmatrix}\right\}\) linearly independent?
    🔗
    Think about it first and see if you can figure it out. Then expand the following solution to check your answer!
    🔗
    Solution.
    These three vectors are linearly dependent since there is a way to write a vector as a linear combination of the other vectors.
    🔗
    Clearly, we can represent the zero vector as a linear combination of the other two vectors with both scalars being zero. That is,
    \begin{equation*} \begin{bmatrix} 0 \\ 0 \\ 0 \end{bmatrix} = 0 \begin{bmatrix} 1 \\ 2 \\ 3 \end{bmatrix} + 0 \begin{bmatrix} 1 \\ 5 \\ 6 \end{bmatrix} \end{equation*}
    Hence, these three vectors are linearly dependent.
    🔗
    🔗
    🔗
    Using the similar argument, we can make sense of the following theorem:
    🔗

    Theorem 5.2.8.

    Any set containing \(\v{0}\) is linearly dependent.
    🔗
    🔗

    Checkpoint 5.2.9.

    When is the set of a single vector linearly independent?
    🔗
    Solution.
    A set containing a single vector \(\set{\v{v}}\) is linearly independent if and only if \(\v{v} \neq \v{0}\text{.}\)
    🔗
    By definition, we are looking at the vector equation \(x_1\v{v} = \v{0}\text{.}\) If \(\v{v}\) is the zero vector, then the scalar \(x_1\) can be any real number (yielding infinitely many solutions), making it a dependent set. If \(\v{v}\) is any non-zero vector, the only way for the equation to hold true is if the scalar \(x_1 = 0\) (the trivial solution), making it an independent set.
    🔗
    🔗
    🔗
    Now let’s look at the next example! Remember the goal is to generalize it!
    🔗

    Example 5.2.10.

    Is \(\left\{\begin{bmatrix} 1 \\ 2 \end{bmatrix}, \begin{bmatrix} -3 \\ 2 \end{bmatrix}, \begin{bmatrix} 1 \\ 6 \end{bmatrix}\right\}\) linearly independent?
    🔗
    Think about it first and see if you can figure it out. Then expand the following solution to check your answer!
    🔗
    Solution.
    These three vectors are linearly dependent since the homogeneous matrix equation is guaranteed to have infinitely many solutions.
    🔗
    Let \(A = \begin{bmatrix} 1 \amp -3 \amp 1 \\ 2 \amp 2 \amp 6 \end{bmatrix}\text{.}\) What does the (reduced) row echelon look like after row reductions?
    🔗
    Well there are only two row so there can be at most two pivot positions.
    🔗
    Yet there are three variables in this equation since each vector gets a mystery scalars. That is, at least one variable is a free variable, and hence the homogeneous equation must have infinitely many solutions.
    🔗
    Therefore, these three vectors are linearly dependent.
    🔗
    🔗
    🔗
    Using the similar argument, we can make sense of the following theorem:
    🔗

    Theorem 5.2.11.

    A set of vectors with more vectors than there are entries in those vectors are linearly dependent.
    🔗
    🔗
    They are the linear dependence tests we can use to quickly determine if a set of vectors is linearly dependent. If you can apply one of the three tests immediately, great! We can conclude that the set is linearly dependent. But the failure to apply any of these tests doesn’t mean the set is linearly independent. It just means we will have to go back to the definition to determine it.
    🔗
    🔗

    Subsection Basis

    Recall that a span of a set of vectors is the smallest subspace containing those vectors, and the span of those vectors contains all possible linear combinations of those vectors. If the spanning set is linearly dependent, then it contains redundant vectors that don’t contribute to the span. We can remove those redundant vectors and still have the same span. For briefness, we only want to keep the necessary vectors that contribute to the span. That is, we want our spanning set to be linearly independent.
    🔗
    This is essentially the basis of a subspace: a linearly independent spanning set that spans the subspace.
    🔗
    That is, the two criteria for a basis are: (1) the set must span the subspace of interest, and (2) the set must be linearly independent. In a more math-y way, we can define a basis as follows:
    🔗

    Definition 5.2.12. Basis of a Subspace of \(\R^n\).

    Let \(U\) be a subspace of \(\R^n\text{.}\) A set of vectors \(\set{\v{v}_1, \v{v}_2, \ldots, \v{v}_p}\) is called a basis of \(U\) if
    1. \(\displaystyle U = \span\set{\v{v}_1, \v{v}_2, \ldots, \v{v}_p}\)
      🔗
      🔗
    2. \(\set{\v{v}_1, \v{v}_2, \ldots, \v{v}_p}\) is linearly independent.
      🔗
      🔗
    🔗
    🔗
    A basis should only include the necessary vectors that contribute to the span.
    🔗

    Example 5.2.13.

    Claim: The set \(\left\{\begin{bmatrix} 1 \\ 0 \end{bmatrix}, \begin{bmatrix} 0 \\ 1 \end{bmatrix}\right\}\) is a basis for \(\R^2\text{.}\)
    🔗
    This claim is true because (1) any vector in \(\R^2\) can be written as a linear combination of these two vectors (so it spans \(\R^2\)) and (2) these two vectors are linearly independent.
    🔗
    Alternatively, \(\left\{\begin{bmatrix} 1 \\ 1 \end{bmatrix}, \begin{bmatrix} 1 \\ 2 \end{bmatrix}\right\}\) is also a basis for \(\R^2\text{.}\) But why... Well this set of vectors also satisfies the two criteria for a basis.
    🔗
    (1) Any vector in \(\R^2\) can be written as a linear combination of these two vectors. Let \(\v{v} = \begin{bmatrix} x \\ y \end{bmatrix}\) be an arbitrary vector in \(\R^2\text{.}\) We want to find scalars \(c_1\) and \(c_2\) such that
    \begin{equation*} c_1 \begin{bmatrix} 1 \\ 1 \end{bmatrix} + c_2 \begin{bmatrix} 1 \\ 2 \end{bmatrix} = \begin{bmatrix} x \\ y \end{bmatrix} \end{equation*}
    Solving this system, we obtain:
    \begin{equation*} \begin{cases} c_1 = 2x - y \\ c_2 = y - x \end{cases} \end{equation*}
    Therefore, any vector in \(\R^2\) can be written as a linear combination of these two vectors, so they span \(\R^2\text{.}\)
    🔗
    (2) These two vectors are linearly independent. The set only contains two vectors and they are not scalar multiples of each other, so they are linearly independent.
    🔗
    Amazing Fact about bases! If \(\set{\v{x}_1, \v{x}_2, \ldots, \v{x}_p}\) and \(\set{\v{y}_1, \v{y}_2, \ldots, \v{y}_k}\) are both bases of a subspace \(U\text{,}\) then \(p = k\text{.}\) In other words, every basis of a given subspace contains exactly the same number of vectors.
    🔗
    So for example, there is no such thing as a basis for \(\R^2\) that contains one or three vectors. They all must contain exactly two vectors!
    🔗
    Note 5.2.14. But Richard... Why is it true?
    Well here is a very interesting approach to prove this fact (and this is also usually the approach to prove two quantities are equal in advanced mathematics!).
    🔗
    Let’s assume that \(\mathcal{B}_1 = \set{\v{x}_1, \v{x}_2, \ldots, \v{x}_p}\) and \(\mathcal{B}_2 = \set{\v{y}_1, \v{y}_2, \ldots, \v{y}_k}\) are both bases of a subspace \(U\text{.}\) Then \(\mathcal{B}_1\) has \(p\) vectors in it and \(\mathcal{B}_2\) has \(k\) vectors in it.
    🔗
    The approach goes like this: we want to show that \(\mathcal{B}_1\) cannot contain more vectors than \(\mathcal{B}_2\) (so \(p \leq k\)), and also that \(\mathcal{B}_2\) cannot contain more vectors than \(\mathcal{B}_1\) (so \(k \leq p\)). Then this will force \(p = k\) to be true!
    🔗
    Since \(\mathcal{B}_1\) is a basis of \(U\) and \(\set{\v{y}_1, \v{y}_2, \ldots, \v{y}_k}\) spans \(U\text{,}\) then \(p \leq k\text{.}\)
    🔗
    And since \(\mathcal{B}_2\) is a basis of \(U\) and \(\set{\v{x}_1, \v{x}_2, \ldots, \v{x}_p}\) spans \(U\text{,}\) then \(k \leq p\text{.}\)
    🔗
    Together, that means \(p = k\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Subsection Dimensions

    Recall that the number of vectors in a basis is a fixed number for a given subspace, and we can generate the whole subspace by taking linear combinations of the vectors in the basis. Then we define the dimension of a subspace as the number of vectors in a basis for that subspace.
    🔗

    Definition 5.2.15. Dimension of a Subspace of \(\R^n\).

    Let \(U\) be a subspace and \(\set{\v{v}_1, \v{v}_2, \ldots, \v{v}_p}\) be a basis of \(U\text{.}\) Then the dimension of \(U\text{,}\) denoted \(\dim U\text{,}\) is the number of vectors in the basis, which is \(p\text{.}\) That is, \(\dim U = p\text{.}\)
    🔗
    🔗

    Example 5.2.16.

    It makes sense to say that the dimension of \(\R^2\) is \(2\) and the dimension of \(\R^3\) is \(3\text{.}\) But how does it match with our definition of dimension?
    🔗
    We can verify this by finding a basis for \(\R^2\) and \(\R^3\) easily. Earlier, we saw that \(\left\{\begin{bmatrix} 1 \\ 0 \end{bmatrix}, \begin{bmatrix} 0 \\ 1 \end{bmatrix} \right\}\) is a basis for \(\R^2\text{,}\) so \(\dim \R^2 = 2\text{.}\) The basis is called the standard basis for \(\R^2\text{.}\)
    🔗
    Similarly, \(\left\{\begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix}, \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix}, \begin{bmatrix} 0 \\ 0 \\ 1 \end{bmatrix}\right\}\) is a basis for \(\R^3\text{,}\) so \(\dim \R^3 = 3\text{.}\) This basis is called the standard basis for \(\R^3\text{.}\)
    🔗
    More generally, the dimension of \(\R^n\) is \(n\) since the standard basis for \(\R^n\) is
    \begin{equation*} \left\{\begin{bmatrix} 1 \\ 0 \\ \vdots \\ 0 \end{bmatrix}, \begin{bmatrix} 0 \\ 1 \\ \vdots \\ 0 \end{bmatrix}, \ldots, \begin{bmatrix} 0 \\ 0 \\ \vdots \\ 1 \end{bmatrix}\right\} \end{equation*}
    and there are exactly \(n\) vectors in this basis.
    🔗
    🔗
    Recall we talked about two fundamental subspaces associated with a matrix in the last section: the null space and the column space. What about their basis and dimension?
    🔗
    Good news! The technique we use to find a spanning set for the null space of a matrix will automatically give us a basis! So we can find a basis of a null space by just doing what we have been doing: solving the homogeneous equation \(A\v{x} = \v{0}\) and writing the solution in parametric form. The vectors corresponding to the free variables will form a basis for the null space.
    🔗
    NOTE: The reason why the vectors corresponding to the free variables will automatically form a basis for the null space is because of the nature of the free variables (talk to Richard if you are super interested in it!). So the only way to end up with the wrong number of vectors in the basis is to have the wrong number of free variables.
    🔗
    Yet the basis for the column space is a bit more tricky to find, as there is no guarantee that the columns of a matrix will be linearly independent. However, we can find a basis for the column space by only including the pivot columns, as these column vectors are guaranteed to be linearly independent (so we essentially take out the free variables to make sure \(A\v{x} = \v{0}\) only has the trivial solution).
    🔗

    Example 5.2.17.

    Let’s say we have the matrix \(A = \begin{bmatrix} -3 \amp 6 \amp -1 \amp 1 \amp 7 \\ 1 \amp -2 \amp 2 \amp 3 \amp -1 \\ 2 \amp -4 \amp 5 \amp 8 \amp -4 \end{bmatrix}\) and this time we want to find a basis for the null space and the column space.
    🔗
    In the last section, we solved the homogeneous equation \(A\v{x} = \v{0}\text{.}\) For our convenience, let’s copy down the reduced row echelon form of the augmented matrix:
    \begin{equation*} \begin{bmatrix} 1 \amp -2 \amp 0 \amp -1 \amp 0 \amp \mid \amp 0 \\ 0 \amp 0 \amp 1 \amp 2 \amp 0 \amp \mid \amp 0\\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    P.S.: Row echelon form is enough as well since all we need to identify are the pivot columns (or not the pivot columns).
    🔗
    Lets’ start with the null space first! The solution to the homogeneous equation \(A\v{x} = \v{0}\text{,}\) in parametric form, looks like this:
    \begin{equation*} \v{x} = \begin{bmatrix} 2x_2 + x_4 \\ x_2 \\ -2x_4 \\ x_4 \\ 0 \end{bmatrix} = x_2\begin{bmatrix} 2 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix} + x_4 \begin{bmatrix} 1 \\ 0 \\ -2 \\ 1 \\ 0\end{bmatrix} \end{equation*}
    where \(x_2, x_4 \in \R\text{.}\) Hence, the null space of \(A\) is
    \begin{equation*} \Nul A = \span\left\{\begin{bmatrix} 2 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix}, \begin{bmatrix} 1 \\ 0 \\ -2 \\ 1 \\ 0\end{bmatrix}\right\} \end{equation*}
    Since these vectors are linearly independent, then a basis for the null space is \(\left\{\begin{bmatrix} 2 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix}, \begin{bmatrix} 1 \\ 0 \\ -2 \\ 1 \\ 0\end{bmatrix}\right\}\) and hence the null space has dimension \(2\text{.}\)
    🔗
    What about a basis for the column space? Let’s stare at the reduced row echelon form of the coefficient matrix part again!
    🔗
    Notice that a pivot position appears in the first, third, and fifth columns. So they do NOT correspond to free variables. Imagine if we remove column 2 and column 4 from the coefficient matrix in the beginning, then we will end up with a homogeneous equation that has no free variable, and hence the columns are linearly independent. Therefore, we can span the column space by just including the first, third, and fifth columns. That is,
    \begin{equation*} \Col A = \span\left\{\begin{bmatrix} -3 \\ 1 \\ 2 \end{bmatrix}, \begin{bmatrix} -1 \\ 2 \\ 5 \end{bmatrix}, \begin{bmatrix} 7 \\ -1 \\ -4 \end{bmatrix}\right\} \end{equation*}
    A basis for the column space is \(\left\{\begin{bmatrix} -3 \\ 1 \\ 2 \end{bmatrix}, \begin{bmatrix} -1 \\ 2 \\ 5 \end{bmatrix}, \begin{bmatrix} 7 \\ -1 \\ -4 \end{bmatrix}\right\}\) and hence the column space has dimension \(3\text{.}\)
    🔗
    Spoiler: You may have noticed that the dimension of the null space and the dimension of the column space add up to the number of columns in the matrix. This is a BIG theorem in linear algebra! We can sense that this is true because we take the columns with pivot positions to form a basis for the column space and use the columns without pivot positions to figure out a basis for the null space. We will focus on this theorem in section 5.4 (and this theorem is a fundamental theorem of linear algebra!) .

    🔗
    Now that we learn more fancy things in linear algebra, we can add a couple more conditions that are equivalent to invertibility:
    • 7. The columns of \(A\) are linearly independent.
      🔗

    • 8. The columns of \(A\) span \(\R^n\text{.}\)
      🔗
      🔗
    • 9. The columns of \(A\) form a basis of \(\R^n\text{.}\)
      🔗
      🔗
    • 10. \(\Col A = \R^n\text{.}\)
      🔗
      🔗
    • 11. \(\dim \Col A = n\text{.}\)
      🔗
      🔗

      🔗
      🔗
    • 13. \(\dim \Nul A = 0\text{.}\)
